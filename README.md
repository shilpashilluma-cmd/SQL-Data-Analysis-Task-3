The objective of this task is to analyze sales data using SQL queries and extract meaningful business insights from a structured dataset.

Tools Used

MySQL (DB Fiddle Online)

GitHub

Dataset

Sample Superstore Sales Data

SQL Concepts Implemented

SELECT statement

WHERE clause for filtering

ORDER BY for sorting

GROUP BY for aggregation

Aggregate functions (SUM, AVG)

Subqueries

CREATE VIEW

Analysis Performed

Retrieved all records from the dataset

Filtered sales data by region

Sorted customer sales in descending order

Calculated total sales by product category

Computed average profit by region

Identified orders with sales above the overall average

Created a view for summarized sales analysis

Key Insights

Certain categories generated higher total sales.

Some regions showed better average profitability.

A few transactions exceeded the overall average sales value.

Outcome

This project demonstrates practical use of SQL for data extraction, aggregation, and business analysis. It highlights the ability to work with structured data and apply core SQL concepts to generate meaningful insights.
